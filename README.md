# Customer Sales Analysis (SQL)

## 📊 Overview
This project analyzes customer purchasing behavior using SQL.

## 🧠 Objectives
- Identify top customers
- Analyze sales by country
- Understand spending patterns

## 🛠️ Skills Used
- SELECT, WHERE
- GROUP BY, HAVING
- JOIN

## 🔍 Key Insights
- Top customers contribute the highest share of revenue
- Sales vary significantly by country
- A small number of customers generate high spending

## 🚀 Conclusion
Customer-level analysis helps businesses identify high-value customers and optimize sales strategies.
